Git使用记录
==============

去跟踪以commit的文件
----------------------

1. ``git rm -f source/*.pyc``
#. ``git cvommit``
#. 在将不要跟踪的文件在 ``.gitignore`` 中设定好


merge的冲突解决
------------------

1. 编辑冲突
#. ``git commit``

    .. note:: 冲突说明
    
        ::
    
          a123
          <<<<<<< HEAD
          b789
          =======
          b45678910
          >>>>>>> 6853e5ff961e684d3a6c02d4d06183b5ff330dcc
          c
       
        其中：冲突标记<<<<<<< （7个<）与=======之间的内容是本地修改，=======与>>>>>>>之间的内容是合入的修改。

        可以使用图形界面工具解决冲突： ::

          git mergetool
